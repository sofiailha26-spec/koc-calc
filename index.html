📚 通用规则(不区分区域)

核心原则:看到主页有风险就判断不合作。账号本身有问题时,整个主页视频可能消失,连带影响品牌形象。

① 违规 / 高风险内容 品牌安全

  • 黄 / 赌 / 毒、明显擦边、软色情、性暗示
  • 暴力血腥、犯罪、美化违法、未成年不当内容
  • 极端言论 / 仇恨 / 歧视
  • 灰产、诈骗、虚假赚钱、博彩引流、烟草毒品赌博导流
  • 频繁引战、辱骂、低俗冲突

② 政治 / 时事敏感 舆情风险

  • 高频讨论政治、国际冲突、社会争议、种族/性别对立
  • 靠热点时事吃流量的评论号
  • 常表强立场、易引发评论区对骂

③ 严重搬运 / 侵权 法律诉讼

  • 大量搬运他人原创视频、整号靠二传/拼接/裁剪
  • 盗用影视片段、综艺、直播切片、Fan Edit
  • 频繁去水印 / 去标识搬运
  • AI 换脸 / 洗稿号(2026 年面临各国法律严打)

④ 明显买量 / 数据异常 无效流量

  • 明显刷播放/赞/互动/粉
  • 合作期间发现买量 → 原则上可不支付

谨慎合作类型 观察

  • 流量不稳定 + 高播放视频互动率 < 0.5% → 需提供 Traffic Source + Average Watch Time + Audience Location 验证
  • 动画类:最好完全原创,或来自公有领域开源素材(Creative Commons / Pixabay / OpenToonz / 早期迪士尼 1920-1940 短片)

停用品类

  • UGC 好物分享、纯搬运、AI 视频、明显买量、造假
  • 沙盒类、影视剧/新闻/八卦搬运、freefire 等中重度游戏(贴 gameplay)、纯 gameplay

发布质量停止合作触发条件

  • 超过 5 条 CPM > 达标 2 倍 / 发布内容低质 / 造假
Memes 与剪辑类同属"二创"大类,打折规则一致(5 折,互动好可不打折)

🎭 Memes

用已有视频/图片/音频 + 幽默夸张元素,带搞笑或讽刺目的。视频标题常含 "funny" "humor" "meme"。

✂️ 剪辑类

对现有素材进行剪辑重组,讲故事或传达主题。如把电影/电视剧片段编辑成新故事。

🤖 AI 创作

使用 AI 自动生成或编辑的内容。如 AI 虚拟人物、自动合成画面/声音。

📥 搬运 vs 剪辑判定

形式判定
原视频 + POV 文案搬运
原视频 + 一句话观点搬运
原视频 + 字幕翻译搬运
原视频 + 箭头 / 圈重点搬运
多片段重剪 + POV 解说剪辑 / 二创

🛍️ 好物分享(不合作)

分享自用产品 / 推荐好物。判断标准:

  • 判断 1:大部分帖子是否 @品牌方
  • 判断 2:评论区是否大量水军(重复人评论无意义内容、隐藏点赞)

报价 7 步法

  1. 问受众 — 确认目标市场匹配
  2. 看爆款空间 — 历史是否有百万观看 + 互动不错的视频
  3. 看互动 — ≥ 4% 最佳;< 1% 不合作或要更多证明
  4. 看发布频率
    • 频繁:7 天内 ≥ 21 条 → 取 7 天最低播放
    • 不频繁:7 天内 ≤ 20 条 → 取最近 20 条最低播放
    • 不足 20 条 → 看主页所有视频流量
    • 超过 1 个月不发布 → 原则不合作
  5. 达人不接受最低观看 → 取第二低观看(报价取整,1万/2万/3万)
  6. 报价时说明:"此价格需团队最终确认,数据每天变化"
  7. 一个订单只能确认一个月合作 — 效果不好需中途暂停

特批条件

互动很好可特批,标准:最近 20 个视频中 ≥ 30% 视频:转发 ≥ 2000 + 评论 ≥ 200

品牌推广标记

  • 所有帖子默认不标记品牌推广
  • 如收到平台提醒或用户举报 → 可特例标记

关于打折

原价 1000 元/条:1 折 = 100 元 · 2 折 = 200 元 · 5 折 = 500 元

询问达人 / 订单情况

审批人需要主页才能给意见,不要只发截图。
  • 分析达人:附 主页链接 + 达人 ID + 昵称 + 订单 ID
  • 订单问题:提供 达人 nickname

特批申请

  • 满足特批标准 → 提审时备注理由
  • 不要私聊审批人 Sofia 问能否额外特批
  • 因不符规则被驳回 → 重复提交一样被驳回
  • 区域化特殊情况 → 充分调研 + 组长沟通后,直接对接塔塔

催审

  • 当前审核时间:工作日 1-3 天内清完
  • 对达人话术:"内部审批 1-3 个工作日"
  • 催审不能加快进度

查看驳回原因

审核后台 → 我发起的 → 查看驳回原因。不要重复提交一模一样的单子。

特殊情况复审流程

  1. 价格复审记录表中填写
  2. 等基建复核确认通过
  3. 回资源库重新提交,备注 "已通过基建复审"
  4. Sofia 在表格确认后审批通过
