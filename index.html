💡 倒推机制:输入目标报价,系统反向计算达人主页"最低播放量"应达到的水平。
用途:谈判时核对达人开价是否合理 / 评估目标合作的达人量级。

📈 互动率计算器

互动率 =（点赞 + 评论 + 转发 + 收藏）/ 播放量 × 100%。
点赞和评论为必填，转发和收藏选填（部分平台不展示）。

💡 支持输入 k 后缀，例如 1.5k = 1,500

📚 通用规则(不区分区域)

核心原则:看到主页有风险就判断不合作。

① 违规 / 高风险内容 品牌安全

  • 黄 / 赌 / 毒、明显擦边、软色情、性暗示
  • 暴力血腥、犯罪、美化违法、未成年不当内容
  • 极端言论 / 仇恨 / 歧视
  • 灰产、诈骗、虚假赚钱、博彩引流

② 政治 / 时事敏感 舆情风险

  • 高频讨论政治、国际冲突、社会争议、种族/性别对立

③ 严重搬运 / 侵权 法律诉讼

  • 大量搬运他人原创视频、整号靠二传/拼接/裁剪
  • 盗用影视片段、综艺、直播切片、Fan Edit
  • AI 换脸 / 洗稿号(2026 年面临各国法律严打)

④ 明显买量 / 数据异常 无效流量

  • 明显刷播放/赞/互动/粉
  • 合作期间发现买量 → 原则上可不支付
Memes 与剪辑类同属"二创"大类,打折规则一致

🎭 Memes

用已有视频/图片/音频 + 幽默夸张元素。视频标题常含 "funny" "humor" "meme"。

✂️ 剪辑类

对现有素材进行剪辑重组,讲故事或传达主题。

🤖 AI 创作

使用 AI 自动生成或编辑的内容。停用品类。

📥 搬运 vs 剪辑判定

形式判定
原视频 + POV 文案搬运
原视频 + 一句话观点搬运
原视频 + 字幕翻译搬运
原视频 + 箭头 / 圈重点搬运
多片段重剪 + POV 解说剪辑 / 二创

🛍️ 好物分享(不合作)

  • 判断 1:大部分帖子是否 @品牌方
  • 判断 2:评论区是否大量水军

报价 7 步法

  1. 问受众
  2. 看爆款空间(百万观看 + 互动不错)
  3. 看互动(≥ 4% 最佳;< 1% 不合作)
  4. 看发布频率
    • 频繁:7 天 ≥ 21 条 → 取 7 天最低播放
    • 不频繁:7 天 ≤ 20 条 → 取最近 20 条最低播放
    • 超过 1 个月不发布 → 不合作
  5. 不接受最低观看 → 取第二低观看
  6. 报价时说明:"需团队最终确认"
  7. 一个订单只能确认一个月合作

特批条件

最近 20 个视频中 ≥ 30% 视频:转发 ≥ 2000 + 评论 ≥ 200

关于打折

原价 1000:1 折 = 100 · 5 折 = 500

询问达人 / 订单情况

  • 分析达人:附 主页链接 + 达人 ID + 昵称 + 订单 ID
  • 订单问题:提供 达人 nickname

特批申请

  • 满足特批标准 → 提审时备注理由
  • 不要私聊 Sofia 问能否额外特批

催审

  • 当前审核时间:工作日 1-3 天
  • 催审不能加快进度

特殊情况复审流程

  1. 价格复审记录表中填写
  2. 等基建复核确认
  3. 回资源库重新提交,备注 "已通过基建复审"
